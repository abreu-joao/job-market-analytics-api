# Job Market API

This project is a simple API designed to collect and analyze job market data related to technology positions.

The goal of the project is to practice backend development and data processing while exploring trends in the tech job market.

## Objectives

- Build a REST API using Python
- Collect and store job-related data
- Provide basic endpoints to access this data
- Practice backend architecture and project organization

## Planned Features

- Job data collection
- Storage of job listings
- Basic statistics about technologies in demand
- API endpoints to retrieve job information

## Technologies

- Python
- FastAPI
- PostgreSQL
- Pandas

## Project Status

This project is currently under development as part of a learning process.
